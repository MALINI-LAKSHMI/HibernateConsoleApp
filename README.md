# Hibernate Console App

The Warehouse Shipment and Dispatch Management System is a console-based Java application developed using Hibernate ORM.
This project is designed to manage warehouse operations such as storing shipment details, dispatching products, tracking
inventory, and maintaining customer and order records.

The system helps in organizing warehouse data efficiently by performing CRUD (Create, Read, Update, Delete) operations 
using Hibernate and relational database mapping.
## Technologies Used
Java

Hibernate Framework

JDBC

MySQL / Oracle Database

Eclipse IDE

Maven (if used)

## Database Operations

This project uses Hibernate to perform:

Insert records

Update records

Delete records

Fetch records using HQL / Criteria
